## 📊 图表制作总体规范

### 技术规格标准
- **分辨率**: 所有图表最低600 DPI，推荐1200 DPI
- **色彩模式**: CMYK (印刷) + RGB (数字)
- **文件格式**:
  - 矢量图: SVG, PDF, AI
  - 位图: PNG, TIFF
  - 双格式输出确保兼容性
- **字体标准**:
  - 主要字体: Helvetica Neue
  - 数学字体: Times New Roman
  - 最小字号: 8pt (确保可读性)
  - 中文标题: 思源黑体

### 色彩设计规范
#### CHI友好色彩方案
```css
/* 主色调 - 色盲安全 */
--primary-blue: #0066CC;      /* 主要数据和重点 */
--secondary-orange: #FF6600;  /* 对比和高亮 */
--accent-green: #009966;      /* 正面结果 */
--warning-red: #CC0000;       /* 警告和问题 */
--neutral-gray: #666666;      /* 辅助文字 */
--light-gray: #F0F0F0;        /* 背景和边框 */

/* 渐变色 - 用于数据可视化 */
--gradient-blue: #E6F2FF → #0066CC;
--gradient-green: #E6FFE6 → #009966;
--gradient-orange: #FFF0E6 → #FF6600;
```

#### 色盲安全性验证
- 使用Coblis色盲模拟器检查
- 确保红绿色盲用户能区分所有颜色
- 提供图案和纹理作为颜色补充

### 布局和排版规范
#### 栅格系统
- **基础栅格**: 12列栅格系统
- **边距**: 统一20px内边距
- **间距**: 元素间距使用8px倍数
- **对齐**: 严格左对齐，图表中心对称

#### 信息层级
```markdown
# H1: 图表标题 - 18pt, Bold
## H2: 子标题 - 14pt, Medium
### H3: 轴标签 - 12pt, Regular
#### H4: 图例 - 10pt, Regular
##### H5: 注释 - 8pt, Regular
```

### 制作流程规范
#### 第1阶段: 数据准备和草图设计
1. **数据验证**: 确保所有数据准确无误
2. **草图设计**: 手绘或数字草图确定布局
3. **色彩方案**: 选择合适的色彩搭配
4. **信息层级**: 确定重点和次要信息

#### 第2阶段: 初稿制作
1. **基础图形**: 创建基本图形和数据展示
2. **标注添加**: 添加必要的文字说明和标签
3. **格式统一**: 应用统一的格式规范
4. **质量检查**: 初步检查技术质量

#### 第3阶段: 精细优化
1. **视觉优化**: 调整颜色、字体、间距
2. **信息优化**: 确保信息传达清晰
3. **可访问性**: 检查色盲安全性
4. **格式输出**: 生成多种格式输出

#### 第4阶段: 最终审核
1. **技术审核**: 检查分辨率、格式、色彩
2. **内容审核**: 确保数据准确和一致
3. **标准审核**: 验证符合CHI标准
4. **批量处理**: 统一处理和命名
