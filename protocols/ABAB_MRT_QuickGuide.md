# ABAB + MRT 实验快速执行指南

## 实验框架
**目标**: 验证「最小生理输入（EMG+GSR）也能支撑具身节律的轻干预」
**设计**: 情境化ABAB + N-of-1微随机 + 事后质化回访
**参与者**: N=12-15（远程/自由职业/设计与研发类为主）
**设备**: 前臂1路sEMG (200Hz) + 指套GSR (4-10Hz)，Mac可视化 + iPhone轻干预

## 研究问题与假设
**RQ**: 在真实工作情境中，仅使用最小生理信号（EMG+GSR），能否支撑不增加负担的节律觉察与自我调节？

**假设**:
- H1（体验）: 干预日的专注时长比对照日↑（≥20%）
- H2（生理）: 干预后GSR峰率下降/EMG RMS区间回落
- H3（主观）: 干预恰当性与不打断感评分更高（Likert 7分≥5.5）

## 实验条件
### ABAB设计
- **A=对照**: 仅记录，不提示
- **B=干预**: 触发轻提示（徽章App：风/水/光微任务），完成即生成徽章

### MRT (Micro-Randomized Trials)
- 在B日里，轻提示按概率p=0.5推送
- 避免机械化提醒，保持自然性

## 实验任务

### Focus任务 (40分钟实际工作)
- 写代码/设计稿/论文排版
- 记录键盘活动、窗口切换
- 保持自然工作状态

### Stress任务 (10分钟限时小任务)
- 复杂重命名/批量改图层
- 播放轻噪音（60-65dB）
- 触发紧张状态

### Restore任务（仅B条件自动触发JITAI）
1. **1分钟自校准**: 呼吸-握拳-放松校准
2. **1-3分钟微任务**: 走到窗边/阳台，完成"风/光/水"任一微任务
3. **20分钟恢复工作**: 回座继续工作

## 采样配置

### 生理信号
- **sEMG**: 200Hz采样，特征：RMS、过零率(ZC)、中值频率(MDF)
- **GSR**: 4-10Hz采样，特征：均值、斜率、峰计数(per min)

### 融合指标 CEI (Combination Embodied Index)
```
CEI = 0.6 * z(RMS_EMG) + 0.4 * z(slope_GSR)
```
其中z为个体化标准分

## 个体化与鲁棒性

### 1分钟自校准流程
1. 静息30秒 → 轻握30秒
2. 获取p10/p90阈值
3. 将全部特征映射到[0,1]区间

### 拒识/弱信号策略
- 信号质量差时转"情境代理"
- 使用屏幕亮度、环境噪声、久坐时长
- 保持交互闭环不断

### 佩戴漂移重置
- 每次开始前做3次握拳-放松
- 更新基线参数

## 行为/系统指标
- **专注时长**: 单应用前台时间、窗口切换次数、键击"爆发长度"
- **中断恢复时间**: 切走后回主任务的秒数
- **干预接受度**: 推送→完成的比例、平均完成时延
