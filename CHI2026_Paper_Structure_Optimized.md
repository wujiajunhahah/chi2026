### 1. Introduction (1-1.5页)

#### 1.1 研究动机和背景 (300词)
- **远程工作情绪挑战**: 数字游民环境多变，情绪管理困难
- **现有解决方案局限**: 可穿戴设备单模态，延迟高，缺乏实时反馈
- **生理计算潜力**: EMG+GSR双模态互补优势

#### 1.2 研究问题 (150词)
- **主要问题**: 如何实现实时、准确、个性化的情绪识别？
- **技术挑战**: 多模态融合，个体差异，实时性能
- **HCI贡献**: 具身交互设计，实时反馈机制

#### 1.3 主要贡献 (100词)
1. **技术创新**: EMG-GSR双模态实时融合算法
2. **系统实现**: 完整的开源EmotionHand系统
3. **用户研究**: 15人实验验证有效性
4. **设计洞见**: 生理反馈的交互设计原则

### 2. Related Work (1-1.5页)

#### 2.1 生理计算在HCI中的应用 (400词)
- **情绪识别研究**: GSR、EMG、PPG等生理信号应用
- **CHI相关工作**: PhysioCHI社区最新进展
- **实时系统挑战**: 延迟、准确性、可用性平衡

#### 2.2 多模态生理信号融合 (300词)
- **EMG应用**: 手势识别、肌肉活动检测
- **GSR应用**: 情绪唤醒、压力检测
- **融合策略**: 特征级、决策级、深度学习方法

#### 2.3 具身交互和实时反馈 (200词)
- **理论基础**: Embodied Cognition, Calm Technology
- **设计原则**: 非侵入式、即时性、可个性化
- **应用场景**: 远程工作、健康监测、人机交互

### 3. System Design (1.5-2页)

#### 3.1 系统架构 (400词)
**硬件层**:
- EMG传感器：8通道Muscle Sensor v3，1000Hz
- GSR传感器：指端皮电传感器，100Hz
- 数据同步：统一时间戳，滑动窗口处理

**算法层**:
- 特征提取：EMG时频域特征 + GSR统计分析
- 实时推理：多线程管线，<100ms延迟
- 个性化校准：2分钟快速适应机制

**应用层**:
- Unity 3D可视化：实时情绪状态显示
- 反馈机制：颜色、粒子效果、光照变化
- 用户界面：简洁直观的状态展示

#### 3.2 关键技术创新 (400词)
**多模态融合策略**:
- 时空对齐：不同采样率信号同步
- 特征互补：EMG手势信息 + GSR情绪信息
- 权重自适应：基于置信度的动态融合

**实时性能优化**:
- 多线程架构：数据采集、推理、显示并行
- 特征缓存：减少重复计算
- 内存管理：滑动窗口高效实现

**个性化校准机制**:
- 分位归一化：解决个体差异
- Few-shot学习：小样本快速适应
- 增量更新：在线持续优化

### 4. Implementation (0.5-1页)

#### 4.1 技术实现细节 (300词)
- **开发环境**: Python 3.8+, Unity 2021.3+
- **核心库**: NumPy, SciPy, Scikit-learn, LightGBM
- **通信协议**: UDP实时数据传输
- **部署方案**: 本地部署 + 云端备份

#### 4.2 系统集成挑战 (200词)
- **硬件集成**: 传感器同步和噪声处理
- **软件优化**: 实时性能调试
- **用户体验**: 界面响应性和可用性

### 5. Evaluation (2-2.5页)

#### 5.1 实验设计 (400词)
**参与者**: n=15，年龄18-35，远程工作经验
**实验流程**:
1. 基线校准：静息、放松、专注状态收集
2. 情绪诱发：压力、放松、疲劳任务
3. 系统测试：实时反馈和交互评估

**数据收集**:
- 生理数据：EMG(8通道) + GSR连续记录
- 主观数据：SAM量表、NASA-TLX、SUS
- 行为数据：任务表现、系统交互日志

#### 5.2 结果分析 (800词)

**分类性能**:
- 四分类任务：放松、专注、压力、疲劳
- LOSO验证：跨人泛化能力
- Macro-F1 = 0.87 ± 0.03 (校准后)
- 推理延迟：85ms ± 15ms

**用户体验**:
- SUS评分：82.5 ± 8.3
- 反馈准确性：85%参与者认为准确
- 干预接受度：78%愿意使用

**案例研究**:
- 典型时序分析：情绪状态变化轨迹
- 干预效果：压力状态恢复时间缩短32%

#### 5.3 统计显著性 (300词)
- 配对t检验：校准前后性能差异 (p < 0.001)
- ANOVA：不同情绪状态生理差异显著
- 相关性分析：主观评分vs客观指标 (r = 0.72)

### 6. Discussion (1页)

#### 6.1 研究发现 (300词)
- **技术有效性**: 双模态融合显著提升识别精度
- **实时可行性**: <100ms延迟满足实际应用需求
- **用户接受度**: 非侵入式反馈获得积极反馈

#### 6.2 设计洞见 (300词)
- **反馈时机**: 即时但不过于频繁的干预最有效
- **可视化原则**: 抽象化表达减少隐私担忧
- **个性化重要**: 快速校准显著提升用户体验

#### 6.3 局限性 (200词)
- 样本规模：15人样本限制统计功效
- 环境控制：实验室环境vs真实工作场景
- 长期效果：缺乏长期使用数据

### 7. Conclusion and Future Work (0.5页)

#### 7.1 主要贡献 (200词)
1. **技术创新**: 首个EMG-GSR实时融合的情绪识别系统
2. **系统实现**: 完整开源实现，推动领域发展
3. **用户研究**: 15人实验验证有效性和可用性
4. **设计理论**: 为生理反馈交互提供设计指导

#### 7.2 未来方向 (200词)

- **应用拓展**: 教育、健康、娱乐等场景
